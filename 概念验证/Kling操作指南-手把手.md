# Kling 2.0 视频生成 · 手把手操作指南

> 每一步都有截图级描述。跟着走就行。

---

## 一、打开 Kling 并登录

1. 浏览器打开 **kling.kuaishou.com**
2. 点右上角 **「登录」**
3. 用 **快手账号** 扫码或手机号登录（没有快手的话，手机号注册一个，30秒）
4. 登录后进入首页，看到顶部有一个 **输入框**，那就是生成视频的地方

> 💡 免费账号每天有 **66 积分**，生成一条5秒视频消耗约 10-20 积分，够跑3-6条。

---

## 二、选择模型和模式

在输入框上方，有一排标签/按钮，按照这个设置：

| 设置项 | 选择 | 原因 |
|--------|------|------|
| **模型** | **Kling 2.0** 或 **2.1**（有哪个选哪个） | 最新模型画质最好 |
| **模式** | **文生视频**（Text to Video） | 我们用文字描述直接生成 |
| **画质** | **高品质** / **专业模式** | 4K可选，1080p也够用 |
| **时长** | **5秒** 或 **10秒**（看镜头需求） | 每个镜头建议先跑5秒测试 |

## 四、设置参数（输入框下方）

| 参数 | 建议值 | 说明 |
|------|--------|------|
| **视频时长** | 5秒或10秒 | 先试5秒看效果 |
| **视频比例** | **9:16**（竖屏） | 抖音格式 |
| **创意相关性** | 默认中间值 | 不要调太高，否则画面会僵硬 |
| **负面提示词**（如果有） | 留空或填 `blurry, distorted face, extra fingers, bad anatomy, watermark, text, logo` | 避免AI生成奇怪的人脸和手指 |

---

## 五、点生成 + 等待

1. 确认提示词和参数无误
2. 点 **「生成」** 按钮（通常在输入框右边，蓝色大按钮）
3. 等待 **30秒 ~ 3分钟**（高峰期可能更久）
4. 生成完成后，页面会显示 **一个视频预览**

> 💡 每次生成会出 **1个版本**。不满意可以再生成一次，同一条提示词多跑几次，挑最好的。

---

## 六、下载视频

1. 鼠标移到生成的视频上
2. 点右下角的 **「⋯」** 或 **下载图标 ⬇**
3. 选择 **「下载」** → MP4 文件保存到本地
4. **重命名文件**：`shot-06.mp4`（按镜头编号命名，方便后续剪辑）

> ⚠️ 重要：下载后立刻改名！不然跑多了分不清哪个是哪个。

---

## 七、不满意怎么办？

| 问题 | 解决办法 |
|------|----------|
| 画面模糊 | 重新生成，或把 `4K` 改成 `8K`，加 `sharp focus, detailed` |
| 人物扭曲/手指变形 | 加负面提示词：`bad anatomy, extra fingers, mutated` |
| 画面和描述不符 | 简化提示词，去掉过于复杂的描述，分成两个镜头 |
| 动作不自然 | 加 `slow motion, cinematic, smooth movement` |
| 色彩不对 | 加具体颜色词：`red glow, dark background, blue sparks` |
| 完全跑偏 | 换一条全新的提示词，或者换 Runway 试试 |
