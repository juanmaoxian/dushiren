## 工作流程

```
第1步：打开 Kling (kling.kuaishou.com) 或 Runway (runway.ml)
第2步：复制上方提示词 → 粘贴 → 生成
第3步：每次生成会出2-4个版本 → 选最好的下载
第4步：下载的MP4统一命名（如 shot-01.mp4, shot-02.mp4）
第5步：全部完成后拖入 DaVinci Resolve 剪辑
```
