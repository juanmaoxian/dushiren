# 《渡世人》30分钟动画 — 制作总体方案

## 一、项目概述

- **原作**：《渡世人》（赛博朋克/科幻）
- **目标时长**：~30分钟（含片头片尾）
- **视觉风格**：赛博朋克 + 废土美学，参考《攻壳机动队》《银翼杀手2049》《赛博朋克2077》
- **色调**：铁锈红、冷蓝灰、霓虹冷光、琥珀金（灰的义眼转变）

---

## 二、制作流程（推荐 AI 辅助管线）

### 阶段 1：剧本改编 ✍️
| 步骤 | 内容 | 工具 |
|------|------|------|
| 1.1 | 小说→动画剧本 | 人工创作（已准备剧本草稿） |
| 1.2 | 对白精修 | Claude / ChatGPT |
| 1.3 | 时长分配 | 按章节划分场次 |

### 阶段 2：视觉设计 🎨
| 步骤 | 内容 | 工具推荐 |
|------|------|----------|
| 2.1 | 角色概念图（5个主要角色） | Midjourney v7 / Flux.1 / Stable Diffusion 3 |
| 2.2 | 场景概念图（灰巢、军工厂、废料场等） | 同上 |
| 2.3 | 关键帧定稿 | Photoshop / Krita 修饰 |
| 2.4 | 色调脚本（Color Script） | 确定每场戏的主色调 |

### 阶段 3：音频制作 🎙️
| 步骤 | 内容 | 工具推荐 |
|------|------|----------|
| 3.1 | 旁白录制 | ElevenLabs / 专业TTS / 真人配音 |
| 3.2 | 角色对白录制 | 同上（不同角色用不同声音） |
| 3.3 | 背景音乐 | Suno AI / Udio / 版权音乐库 |
| 3.4 | 音效制作 | Artlist / Epidemic Sound / 自行录制 |

### 阶段 4：视频生成 🎬（核心环节）
| 方案 | 工具 | 适合场景 | 成本估算 |
|------|------|----------|----------|
| **A: AI视频生成** | Runway Gen-4 / Kling 2.0 / Wan2.1 / Sora | 所有场景 | $100-500/月订阅 |
| **B: AI图片+动画化** | Midjourney → Runway/Pika/Luma 图生视频 | 抒情慢镜 | $60-200/月 |
| **C: 3D引擎实时渲染** | Unreal Engine 5 + MetaHuman | 角色对白、动作戏 | 免费+学习成本 |
| **D: 混合方案（推荐）** | 3D引擎做角色 + AI做背景 + 传统合成 | 全片 | 综合以上 |

> **2026年推荐路径**：如果追求效率，用 **Wan2.1/Kling 2.0 直接文生视频** 为主；关键抒情段落用 Midjourney出图→Runway图生视频。如果追求品质+有技术基础，用 **Unreal Engine + AI辅助**。

## 八、重要提示

1. **AI视频生成的局限性**：目前AI视频生成工具（Runway/Kling/Wan2.1等）在角色一致性、长镜头稳定性上仍有不足。建议：
   - 重要角色对话场景：考虑3D引擎或多次迭代
   - 氛围/空镜场景：AI非常适合
   - 动作场景：AI + 快速剪辑掩饰不稳定性

2. **叙事驱动**：赛博朋克动画的成败在于氛围。这部小说本身有非常强烈的视觉意象——铁锈色的天空、霓虹灯、半张机械脸、红色义眼——这些是做视觉的绝佳素材。

3. **迭代心态**：30分钟的动画不需要一次完美。先做一个5分钟的精华片段作为"概念验证"，然后逐步扩展。
