# 《渡世人》角色设定与分镜脚本

---

## 一、角色设定

### 1. 灰（Grey）— 主角

**基本信息**
- 年龄：26岁
- 型号：A-7"渡鸦"型改造人 · 渡鸦1号
- 国籍：中国
- 现状：灰巢废料场维修工

**外观特征**
- **面部**：脸上有许多伤痕，透出金属的光泽——机械骨架埋在皮下，仅在伤痕深处隐约可见钛合金冷光
- **右眼**：正常为黑瞳，植入式光学传感器。感受到压力或进入战斗模式时变为暗红色
- **左眼**：生物学眼球，正常黑色瞳孔
- **身体**：左臂在战争中失去，左肩处为金属接口，偶尔冒火花
- **皮肤**：苍白、粗糙，有细小伤疤
- **头发**：黑色卷发，扎起
- **穿着**：浅棕色工装服，多处磨损、油渍和修补痕迹
- **靴子**：黑色靴子
- **身高**：高挑，约172cm
- **身材**：玲珑有致
- **体态**：行动时有军人的精准，微而不显

**性格**：冷脸萌妹 — 面瘫脸，反应慢半拍，但对上眼就绷不住

**视觉关键词**：伤痕透金属光、黑色卷发扎起、浅棕工装、黑靴、暗红右眼

**义眼特效**：
- 日常状态：黑色瞳孔
- 压力/战斗模式：暗红色光芒
- 扫描模式：红光变亮，视野色调变冷蓝
- 情绪波动：红光闪烁频率增加
- 回放记忆：快速闪烁，像读取数据
- **关键转变时刻**：暗红色→琥珀色光芒亮起（故事核心视觉符号）

---

### 2. 老洪（Hong Zheng）— 导师/创造者

**基本信息**
- 年龄：60+岁
- 原名：洪铮
- 过去：联邦AI研究院首席工程师，A-7项目总设计师
- 现状：灰巢废品店老板

**外观特征**
- **面部**：沟壑纵横的皱纹，饱经风霜。眼神锐利但常刻意显得迟钝
- **腿**：左腿在战争中失去，用一根实木棍代替——木腿敲铁皮地面：一轻一重
- **头发**：花白，稀少
- **手**：老年人的手，但手指仍有工程师的精准（修精密仪器时可以看到）
- **穿着**：破旧工装，和灰一样满是油渍
- **身高**：年轻时应该很高，现在微微驼背

**视觉关键词**：木腿、皱纹、藏在迟钝下的锐利、台灯下的半张脸

**关键道具**：
- 木腿（走路的节奏感是声音设计的重要元素）
- 老旧的台灯（总是照着半张脸）
- 铁盒子（藏在柜台最底层抽屉里）
- 加密终端（三十年前的旧设备）

---

### 3. 螺丝（Screw）— 黑客少女

**基本信息**
- 年龄：13岁
- 原名：未完全透露（陆醒的妹妹）
- 身份：灰巢黑市黑客
- 哥哥：陆醒

**外观特征**
- **身高**：140cm（很小只）
- **发型**：乱糟糟的深色头发，扎一个歪马尾
- **面部**：大眼睛，眼底有黑眼圈（熬夜太多），脸上经常蹭到机油或巧克力。眼神不像小孩——太锐利、太警觉
- **穿着**：一件明显太大的大号工装外套（很可能就是她哥哥陆醒留下的），袖口挽了好几道
- **体态**：常蹲在椅子上而不是坐着，像一只警觉的小动物
- **无改造痕迹**：完全的人类，这在灰巢反而少见

**视觉关键词**：大外套、歪马尾、黑眼圈、蹲椅子、破笔记本

**关键道具**：
- 拼装的破笔记本（贴满了贴纸和胶带）
- 满地数据线和空罐头

---

### 4. 布洛（Buro）— 赏金猎人/曾经的同伴

**基本信息**
- 年龄：和灰相仿，30+岁
- 型号：A-7改造人（和灰同型号）
- 现状：赏金猎人（实则为黑牙工作）

**外观特征**
- **改造度**：全身90%改造——比灰更完整、更新。钛合金装甲皮肤（第二层）
- **人类部分**：只剩右脸颧骨以下到右下巴一小块人类皮肤。右眼下面有一条陈旧的刀疤——那是他最显眼的人类特征
- **身体**：双腿、双臂、躯干、双手、胸腔——全改。机械躯体线条冷硬
- **义眼**：红色（但在最后场景中转向琥珀色）
- **声音**：很平，几乎不带感情，直到被灰打破
- **穿着**：深色战术装备，赏金猎人的实用风格

**视觉关键词**：几乎全身机械、眼角刀疤、人类皮肤的最后一小块

**动作特征**：
- 走路时机械腿发出沉稳的金属声——当、当、当
- 手部动作极其精准，但偶尔会不自觉地松开（人性残留的微表情）

---

### 5. 陆醒（Lu Xing）— 不该拿武器的人

**基本信息**
- 年龄：牺牲时20+岁
- 身份：渡鸦小队技术军医，老洪最得意的学生
- 状态：已故（仅在回忆/影像中出现）

**外观特征**
- **面部**：年轻、戴眼镜（厚厚镜片的那种），总在笑
- **穿着**：白大褂（实验室）/ 军装（前线）——都不太合身，军装穿得不太像个军人
- **体态**：书生气，在前线显得格格不入
- **关键特征**：眼镜经常歪了也不知道

**视觉关键词**：眼镜、白大褂、笑、"唯一没拿过武器的人"

**关键场景**：
- 三十年前照片里：后排，戴着眼镜，穿着白大褂而不是军装
- 最后的影像日志里：眼镜歪着，身后是燃烧的村庄

---

### 6. 少年A-7（配角/对手）

**外观特征**
- **年龄**：15-16岁
- **改造**：崭新的金属四肢，接口处螺丝还亮着崭新的银光
- **面部**：脸上有刚缝合的伤口，义眼发着不稳定的、可怕的红色
- **身高**：不到170cm
- **状态**：系统未校准，只会执行"任务未完成"
- **嘴里反复念着**："任务未完成。任务未完成。"

**视觉关键词**：崭新的改造体（与灰的旧形成对比）、不稳定的红光

---

### 7. 黑牙（Black Fang）— 反派（不露面）

- 仅通过档案照片和投影出现
- 中年男人，精致的机械手表，衣冠楚楚
- 前联邦军方AI采购负责人
- 代表"体制的冷酷"

## 二、场景设定

### 1. 灰巢（Ash Nest）— 垃圾行星
- **天空**：永远的铁锈色雾层。没有星星。没有真正的正午
- **太阳**：衰老的红矮星，挂在轨道外侧，懒洋洋的
- **地表**：无尽的废料场——废弃飞船骨架、电路板山丘、生锈金属结构
- **光线**：只有两种光——灰蒙蒙的日光，和霓虹灯管的冷光
- **色调**：铁锈红 + 灰褐 + 霓虹冷蓝/紫

### 2. 灰的维修室（集装箱铁皮屋）
- 四面铁墙上挂满零件和工具
- 一张铁架子床+破床垫
- 天花板上有一道裂缝（可以看到外面）
- 一面破镜子
- 唯一的光源：灰的义眼红光

### 3. 老洪的废品店
- 昏暗。铁皮地面
- 柜台后面一盏老旧的台灯
- 堆满旧电器和零件
- 最底层抽屉里的铁盒子

### 4. 旧军工厂（灰巢下层）
- 完全黑暗。被废弃三十年
- 走廊很长，两侧是废弃实验室和车间
- 油漆剥落，露出混凝土
- B区地下三层：钛合金防爆门，主服务器

### 5. 螺丝的地下基地
- 挤满设备的小房间
- 墙上十几块显示屏
- 满地数据线和空罐头
- 冷蓝色屏幕光照亮一切

## 四、色彩脚本（Color Script）

| 章节 | 主导色调 | 关键色 |
|------|----------|--------|
| 片头 | 铁锈红 + 黑色 | 暗红、深灰 |
| 第一章·锈 | 暗灰 + 铁锈红 + 霓虹冷蓝 | 灰的义眼红光 |
| 第二章·信号 | 暗灰 + 电子冷蓝（信号相关） | 蓝色闪烁 |
| 第三章·潜入者 | 冷蓝 + 暗绿（夜视）+ 蒸汽白 | 螺丝屏幕的蓝光 |
| 第四章·43秒 | 冷灰 + 暖火橙（闪回）→ 逐渐过渡 | 布洛的冷金属 |
| 第五章·真相 | 暖白（回忆）+ 火光→ 琥珀金 | **琥珀色首次出现** |
| 43秒段落 | 火光橙 + 爆炸白 → 定格 | 铭牌的金色 |
| 第六章·渡世 | 琥珀金 + 温暖铁锈 | **红色→琥珀的转变完成** |
| 三个月后 | 温暖的灰 + 金色余晖 | 温暖色调 |
| 彩蛋 | 暗 → 琥珀色光 | 希望的颜色 |
